Staying up late, jobless and wanted to read the NASM documentation. And so I write the following script...

[sourcecode language="bash"]
#!/bin/bash
var="http://www.nasm.us/doc/nasmdoc"
no="0"
ext=".html"
while [ $no -lt 10 ]
do
wget $var$no$ext
no=$((++no))
done
[/sourcecode]

The rest were manually downloaded :P
