And for some reason emerge wouldn't open as a mergetool when I followed the steps mentioned at Emacs Wiki. I kept getting the following error:

[code language="bash"]The merge tool emerge is not available as '/Users/prometheansacrifice/bin/emerge-for-git[/code]

Opened docs at git-scm and realized git passes names of the local, remote and base copies of the file with merge conflicts with $LOCAL, $GLOBAL and $REMOTE respectively. The filename of the final merged copy is $MERGED.Â  So all you need to do is,

[code language="bash"]git config --global mergetool.emerge.cmd 'emacs -Q $LOCAL $REMOTE $BASE $MERGED'[/code]

And voila! You're ready to go.
